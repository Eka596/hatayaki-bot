# Hatayaki
Многофункциональный Discord-бот для управления сервером и общения с ИИ-персонажем. Написан на JavaScript с использованием Discord.js и LM Studio.

## Что умеет
- **Ролевое общение с ИИ:** отвечает в стиле заданного персонажа (промпт настраивается).
- **Скачивание вложений:** сохраняет файлы из сообщений.
- **Разделение команд по ролям:** разные уровни доступа для администраторов и пользователей.

## Технологии
- JavaScript (Node.js)
- Discord.js v14
- qwen2.5-vl-7b (LM Studio)
- Работа с файловой системой (fs)

## Команды
### Для всех участников
- `!me` — моя статистика (сколько и каких файлов загружено)
- `!top` — топ-5 участников сервера по числу сохранённых вложений
- `!hug [@user]` — обнять участника (или всех)
- `!pat [@user]` — погладить

### Для администраторов (и владельца)
- `!watch #канал` — начать сохранять вложения из канала
- `!unwatch #канал` — перестать сохранять вложения
- `!list` — показать отслеживаемые каналы
- `!chat watch #канал` — разрешить боту отвечать в канале
- `!chat unwatch #канал` — запретить боту отвечать
- `!chat list` — список каналов, где бот отвечает
- `!reloadchat` — перезагрузить JSON-фразы (если используется резервный режим)

### Только для владельца бота
- `!reload` — перезагрузка основных модулей без остановки
- `!debug` — переключение режима логирования (debug / production)
- `!setting <ключ> <значение>` — изменить настройку (`prefix`, `cooldownSeconds`, `phrasesPath`)
- `!gif add <hug/pat> (прикрепите GIF)` — добавить гифку в коллекцию
- `!gif remove <hug/pat> <ID>` — удалить гифку
- `!gif list <hug/pat>` — посмотреть список гифок

## ⚙️ Установка
1. **Скачайте проект** или склонируйте репозиторий.
2. **Установите зависимости**:
   ```bash
   npm init -y
   npm install discord.js dotenv sqlite3 axios
3. **Создайте файл .env на основе .env.example и заполните своими данными**:
   ```env
   TOKEN=ваш_токен_бота
   OWNER_ID=ваш_discord_id
   LOG_MODE=debug
   AI_MODE=ai
   AI_API_URL=http://localhost:1234/v1/chat/completions
   AI_MODEL=qwen2.5-vl-7b
4. **Установите LM Studio** с ([официального сайта](https://lmstudio.ai/)), скачайте **модель qwen2.5-vl-7b**
(или любую совместимую) и запустите локальный сервер на вкладке Local Server.
5. **Запустите бота**:
   node index.js
